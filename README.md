## :checkered_flag: Starting

```bash
# Clone this project
$ git clone https://github.com/olamilekan21/iconsax

# Access
$ cd iconsax

# Install dependencies
$ yarn

# Run the project
$ yarn start

# The server will initialize in the <http://localhost:3000>
```
